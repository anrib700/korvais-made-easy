Use of parentheses to represent Karvai

A number or Sollu within parentheses denotes a Karvai for that duration. Simple examples are given in table 6.

Table 6: Karvai represented using parentheses
Representation Sollu
(1) (Tha)
(2) (Tha,)
(3) (Tha;)
(4) (Tha,;)
(5) (Tha;;)
